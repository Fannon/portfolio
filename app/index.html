Example Projects

isomatic

Isomatic is an online tool to generate isotype graphics in the browser. The user can copy and paste the data and adjust several options. The result can be exported (and reimported) to SVG.

  • HTML5/JavaScript App. Using Backbone.js, D3.js, Zurb Foundation, etc.
  • My part was programming the visualisation and the User-Interface
  • www.isomatic.de
  • GitHub Open Source
  • 2013 - 2014

SVG Generator

This is an online Tool that helps you create random SVG Images in many ways. The result can be exported to SVG Vector Graphic Files

  • JavaScript App. Using Raphael.js SVG Library
  • www.svg-generator.de
  • GitHub Open Source
  • 2011

MediaWiki Benchmark Tool

This Webtool lets you benchmark MediaWiki Installations. The resulting Graphic and Data can be exported to several formats.

  • JavaScript App. Using Google Charts and Twitter Bootstrap
  • www.fannon.de/smw/mediawiki-benchmark/
  • GitHub Open Source
  • 2013

Obacht Game

Obacht is a cross plattform multiplayer Game for mobile Devices and Webbrowsers. Available on the Apple Store and Google Play Store.

  • HTML5/JavaScript Game. Using Lime.js as Game-Engine and Node.js as Multiplayer-Server
  • My part was programming the Multiplayer-Server and code architecture in general.
  • www.obacht-game.com
  • GitHub Open Source
  • 2013
